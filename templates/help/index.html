Connexion

Comment se connecter ?
  1. Accédez à la page de connexion
  2. Entrez votre nom d'utilisateur
  3. Entrez votre mot de passe
  4. Cliquez sur "Se connecter"
Si vous avez oublié votre mot de passe, contactez votre administrateur.

Gestion des dossiers

Actions disponibles
Créer un dossier
  1. Cliquez sur "Nouveau dossier"
  2. Remplissez tous les champs obligatoires
  3. Cliquez sur "Enregistrer"
Rechercher un dossier
  1. Utilisez la barre de recherche
  2. Filtrez par service ou statut
  3. Cliquez sur un dossier pour voir les détails

Questions fréquentes

  1. Ouvrez le dossier concerné
  2. Cliquez sur "Transférer"
  3. Sélectionnez le service de destination
  4. Ajoutez un commentaire si nécessaire
  5. Confirmez le transfert

  1. Ouvrez le dossier à archiver
  2. Cliquez sur "Archiver"
  3. Confirmez l'archivage
Un dossier archivé peut toujours être consulté mais ne peut plus être modifié.
